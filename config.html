The config file

Back to index

Each TShock installation automatically generates a configuration file which can edit basic settings when the server starts.

The file "config.json" is located in the tshock folder, in the same directory as TerrariaServer.exe.

Being a JSON file, it is extremely critical that you edit the file in a standard text editor. Use Notepad++ to edit your file, or another text editor used for programming. Before restarting TShock, check the syntax of your file using JSONLint, and verify that it contains no errors.
